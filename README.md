# 🎓 Student Manager C

Student Manager C 是一个使用 C 语言编写的简单的学生管理系统。

## ✨ 主要功能

- **添加学生**: 添加新的学生信息。
- **显示所有学生**: 显示所有学生的信息。
- **查找学生**: 按学号查找学生。
- **删除学生**: 按学号删除学生。
- **数据持久化**: 支持将学生数据保存到文件，并从文件中加载。
- **排序**: 支持按学号或成绩对学生进行排序。

## 🛠️ 如何构建和运行

### 1. 克隆仓库

```bash
git clone https://github.com/SaintFore/student_manager_c.git
```

### 2. 编译

你可以使用任何 C 编译器来编译这个项目。例如，使用 gcc：

```bash
gcc main.c student.c -o student_manager
```

### 3. 运行程序

```bash
./student_manager
```

## 🚀 使用方法

程序启动后，你会看到一个菜单，你可以根据菜单的提示进行操作。

```
--- 学生管理系统 ---
1. 添加学生
2. 显示所有学生
3. 按学号查找学生
4. 按学号删除学生
5. 存储数据
6. 读取数据
7. 按学号排序
8. 按成绩排序
0. 退出程序
----------
请输入你的选择:
```
